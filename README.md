# FancyHelperReloadService
为FancyHelper而生的重载工具，仅用于重载FancyHelper

---

自 https://github.com/baicaizhale/FancyHelper/commit/6f60aaf5ed788890d2da4a4abb9a179146050bfc 起，FancyHelper不再使用PlugManX进行重载。
为此，我们开发了FancyHelperReloadService来帮助FancyHelper重载。
我们继续开源此插件。您完全可以审查和贡献此仓库。
不过FancyHelperReloadService除了重载没有任何功能，应该是不会再更新了。
